# Simple_Stack

这是一个基于Vue3 + Express.js + MongoDB + nginx开发的轻量级的全栈应用框架，使用Docker容器化部署。

## 系统要求
- Docker
- Docker Compose

## 快速启动

1. 克隆仓库：
```bash
git clone https://github.com/YuanYiShuo7/Simple_Stack.git
```
2. 启动服务:
```bash
docker-compose up --build
```
3. 访问应用：
- 客户端：http://localhost
- 服务端：http://localhost:5000

4. 停止服务：
```bash
docker-compose down
```
